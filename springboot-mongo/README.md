# Mongo整合springboot

- [github地址-spring-data-mongodb](https://github.com/spring-projects/spring-data-mongodb)

- pom.xml 

    ```xml
    <!--添加mongo 相关的maven依赖-->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-mongodb</artifactId>
    </dependency>
    
    ```
